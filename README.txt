ΠΕΡΙΓΡΑΦΗ
------------------------------------------------------------------------------
Το DatabaseZooProject είναι μια εφαρμογή γραμμένη σε Python, η οποία χρησιμοποιεί
το terminal ως UI, σχεδιασμένη για τη διαχείριση και ανάλυση δεδομένων ενός Ζωολογικού Κήπου. 
Παρέχει μια διεπαφή για αλληλεπίδραση με τη βάση δεδομένων, 
εκτέλεση ερωτημάτων (queries) και παραγωγή στατιστικών αναφορών σχετικά με 
τη λειτουργία του πάρκου.

ΑΡΧΕΙΑ ΣΤΟ REPOSITORY
------------------------------------------------------------------------------
* main.py             : Το σημείο εκκίνησης της εφαρμογής. Χειρίζεται τον 
                        κύριο βρόχο εκτέλεσης και τη λογική υψηλού επιπέδου.
* connectToDB.py      : Διαχειρίζεται τη σύνδεση με την εξωτερική βάση 
                        δεδομένων. Περιέχει το File Path του αρχείου της βάσης (.db)
			με την οποία επικοινωνεί η εφαρμογή καθώς και το αντικείμενο 
			σύνδεσης.
* queries.py          : Περιέχει τη συνάρτηση εκτέλεσης των Queries ενώ ταυτόχρονα 
			αποθηκεύει τα ερωτήματα SQL που χρησιμοποιεί η 
                        εφαρμογή για την ανάκτηση χρήσιμων πληροφοριών.
* statistics.py       : Περιέχει τη λογική για τον υπολογισμό και την 
                        εμφάνιση στατιστικών στοιχείων (π.χ. πλήθος ζώων, 
                        πωλήσεις εισιτηρίων κ.λπ.).
* terminalFeatures.py : Συναρτήσεις που διαχειρίζονται τις εισόδους του χρήστη
			και επικοινωνούν με τις εκάστοτε βαθύτερες συναρτήσεις 
			λογικής της κάθε επιλογής.

ΠΡΟΑΠΑΙΤΟΥΜΕΝΑ
------------------------------------------------------------------------------
1. Εγκατεστημένη Python 3 στον υπολογιστή σας.
2. Το .db αρχείο που περιέχει τη βάση του ζωολογικού κήπου.
3. Βιβλιοθήκες σύνδεσης Python (Python Database Connectors).

ΕΓΚΑΤΑΣΤΑΣΗ & ΡΥΘΜΙΣΗ
------------------------------------------------------------------------------
1. Κλωνοποίηση του repository(Clone) μέσω Terminal/CMD:
   $ git clone https://github.com/EMMANOUIL-ALKIS-SOURIS/DatabaseZooProject.git

   Ή εναλλακτικά αποθηκεύστε τα αρχεία σε έναν φάκελο και τρέξτε το main.py

2. Εγκατάσταση Εξαρτήσεων:
   Εγκαταστήστε μέσω pip:
   $ pip install sqlite3

3. Ρύθμιση Βάσης Δεδομένων: 
   Στο connectToDB.py αρχείο καθορίστε το file path του αρχείου .db της βάσης 
   του ζωολογικού κήπου που θέλετε να διαχειριστείτε


ΟΔΗΓΙΕΣ ΕΚΤΕΛΕΣΗΣ
------------------------------------------------------------------------------
Για να ξεκινήσετε την εφαρμογή, ανοίξτε το cmd, πλοηγηθείτε στον φάκελο που 
έχετε αποθηκεύσει όλα τα .py αρχεία και εκτελέστε την παρακάτω εντολή:  

   $ python main.py

Εναλλακτικά χρησιμοποιώντας κάποιο code editor/IDE πχ Visual Studio Code 
εκτελέστε το αρχείο main.py.

Ακολουθήστε το μενού που εμφανίζεται για να πλοηγηθείτε στις διάφορες λειτουργίες.

ΛΕΙΤΟΥΡΓΙΕΣ (FEATURES)
------------------------------------------------------------------------------
* Συνδεσιμότητα Βάσης: Ασφαλής διαχείριση σύνδεσης με τη βάση του ζωολογικού κήπου.
* Εκτέλεση Ερωτημάτων/ Queries: Προκαθορισμένα ερωτήματα για ανάκτηση βασικών πληροφοριών.
* Στατιστικά: Μονάδα ανάλυσης για την οπτικοποίηση ή αναφορά τάσεων δεδομένων.
* Τερματικό UI: Φιλικό προς τον χρήστη περιβάλλον γραμμής εντολών για εύκολη πλοήγηση.
